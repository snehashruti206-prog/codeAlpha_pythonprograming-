# CodeAlpha Python Programming Internship

This repository contains the projects completed during my Python Programming Internship at CodeAlpha.

## Projects Included:

### 1. 🎮 Hangman Game
A classic word-guessing game built using Python. The program selects a random word, and the player tries to guess it by suggesting letters within a certain number of attempts.

### 2. 🤖 Basic Chatbot
A simple rule-based interaction bot designed using Python conditional logic. It can reply to basic greetings, questions, and exit commands smoothly.

## Technologies Used:
* Python 3
* Random module (for Hangman)
